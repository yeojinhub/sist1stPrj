## 🖥️ 개요
> 프로젝트 이름: 베스트캠퍼스
> 
> 본 프로젝트는 학원 관리 시스템 구축을 목적으로 하며, 학생, 강사, 관리자를 위한 3가지 프로그램을 개발하는 것이 목표입니다.
> 
> 각 사용자 유형별로 권한과 기능이 상이하나, 공통된 데이터베이스를 기반으로 상호 연동되는 구조를 갖추고 있습니다.
